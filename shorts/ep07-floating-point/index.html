Bukan 0.3? WTF
Kenapa? Binary.
Komputer menyimpan desimal dalam biner. Beberapa angka yang "sederhana" di desimal menjadi pecahan tak terhingga di biner.
1 ÷ 3 (desimal)
0.333333…
0.1 (biner)
0.000110011…
Pakai Decimal.
Untuk aplikasi finansial — presisi wajib.
Python
from decimal import Decimal
Decimal("0.1") + Decimal("0.2")
Java
new BigDecimal("0.1")
  .add(new BigDecimal("0.2"))
JavaScript
import Decimal from "decimal.js"
new Decimal("0.1").plus("0.2")
